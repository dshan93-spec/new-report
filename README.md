# Perovskite Red LED Executive Report

GitHub Pages 배포용 정적 사이트 패키지입니다.

## 배포 방법

1. GitHub에서 새 저장소 생성 예: `microled-report`
2. 이 폴더 안의 파일들을 저장소 루트에 업로드
   - `index.html`
   - `README.md`
   - `.nojekyll`
3. 저장소에서 **Settings → Pages** 이동
4. **Build and deployment → Source: Deploy from a branch** 선택
5. **Branch: main / root** 선택 후 Save
6. 배포 주소 예:

```text
https://dshan93.github.io/microled-report/
```

## 파일 설명

- `index.html`: 한 장짜리 경영진 보고서
- `.nojekyll`: GitHub Pages에서 정적 파일을 그대로 서비스하기 위한 설정
- `README.md`: 배포 안내
