Perovskite Red LED는 높은 EQE와 좁은 FWHM으로 차세대 후보 가치는 크지만, 고휘도 장기수명·ion migration·halide segregation·패터닝 균일성·Pb 대응이 아직 제품 리스크입니다. 단기 제품 적용은 OLED/III-V 유지, Perovskite는 gated option bet로 관리하는 것이 합리적입니다.

3. 무기물인데 왜 신뢰성이 약한가?

Soft lattice

GaN/AlGaInP처럼 강한 공유결합 단결정이 아니라, 상대적으로 부드러운 이온성 결정입니다. 열과 전기장에 구조가 민감합니다.

Ion migration

I⁻, Br⁻, vacancy가 전기장 아래에서 이동하여 계면 반응, 결함 증가, 밝기 저하를 유발합니다.

Color drift

혼합 halide 조성에서는 I-rich / Br-rich 영역 분리가 생겨 peak wavelength와 색좌표가 움직일 수 있습니다.

Moisture/heat

수분과 열에 의해 결정구조 변화 또는 분해가 가속되며, 장기수명 확보에 불리합니다.

4. 의사결정 Gate

Gate 1: MaterialPeak EQE ≥25%, 순적색, FWHM <30 nm, color drift 내부 기준 통과
Gate 2: Device1000 cd/m² 이상 장기 T50/T95, 고전류 안정성, encapsulation 통과
Gate 3: Panelpixelated AMBP integration, 균일도, 패터닝 수율, Pb 규제 package 확보
